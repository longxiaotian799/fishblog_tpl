## 方案一: 安装JDK 8 (推荐)

### 1. 下载JDK 8

**国内镜像（推荐）:**
- 华为镜像: https://repo.huaweicloud.com/java/jdk/8u202-b08/jdk-8u202-windows-x64.exe
- 或 Oracle官方: https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html

### 2. 安装JDK 8

1. 运行安装程序
2. 选择安装路径，例如: `C:\Program Files\Java\jdk1.8.0_202`
3. 完成安装

### 3. 设置环境变量

**方法A: 使用系统设置**
1. 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
2. 在"系统变量"中:
   - 新建变量 `JAVA_HOME_8`，值为你的JDK 8路径 (如 `C:\Program Files\Java\jdk1.8.0_202`)
   - 编辑 `Path` 变量，将 `%JAVA_HOME_8%\bin` 移到Java 26路径之前

**方法B: 使用脚本临时切换（当前命令行窗口有效）**
```cmd
set JAVA_HOME=C:\Program Files\Java\jdk1.8.0_202
set PATH=%JAVA_HOME%\bin;%PATH%
java -version
```

### 4. 启动项目

安装并配置好JDK 8后，运行:
```cmd
cd C:\Users\31847\fishblog_tpl
start-all.bat
```

## Docker容器管理

### 查看状态
```cmd
docker ps --filter "name=fishblog"
```

### 停止所有容器
```cmd
docker stop fishblog-mysql fishblog-redis fishblog-rabbitmq fishblog-es fishblog-backend
```

### 启动所有容器
```cmd
docker start fishblog-mysql fishblog-redis fishblog-rabbitmq fishblog-es
# 如果使用Docker后端:
docker start fishblog-backend
```

### 删除所有容器（重新初始化）
```cmd
docker rm -f fishblog-mysql fishblog-redis fishblog-rabbitmq fishblog-es fishblog-backend
```

## 常见问题

### Q: Maven下载依赖慢？
A: 已配置阿里源，确保使用 `D:\Download\apache-maven-3.9.15-bin\apache-maven-3.9.15\bin\mvn`

### Q: 端口被占用？
A: 修改对应的配置文件或停止占用端口的程序
